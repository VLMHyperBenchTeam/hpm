# Задача: Реализация Implication Merging

## Контекст
Необходимо реализовать логику слияния параметров зависимостей, описанную в плане `hsm/docs/plans/implication_merging.md`.

## Шаги реализации
1.  **Models**: Обновить Pydantic-модели в `hsm/src/hyper_stack_manager/models.py` для поддержки параметров в `implies`.
2.  **Core**:
    *   Изменить метод `_resolve_container_config` (или создать новый), чтобы он принимал параметры от пакета.
    *   Реализовать логику группировки и слияния параметров в `HSMCore.sync`.
3.  **Interpolation**: Добавить поддержку `${HSM_MERGED_PARAMS.key}` в логику обработки переменных окружения.

## Критерии готовности
*   При выборе двух пакетов, требующих один и тот же контейнер с разными параметрами, HSM генерирует один сервис с объединенной конфигурацией.
