# Идея: Интерактивный CLI (TUI) для HPM

**Статус**: Предложено
**Дата**: 2026-02-02

## Концепция

По мере роста HPM управление сложными конфигурациями групп через флаги командной строки (`--option`) становится утомительным. Интерактивный текстовый интерфейс (TUI) значительно улучшит Developer Experience (DX).

## Функциональность

1.  **Group Explorer (Навигатор по группам)**:
    *   Список всех доступных групп из реестра.
    *   Визуальные индикаторы для активных/выбранных групп.
    *   Раскрытие групп для просмотра доступных опций (реализаций).

2.  **Smart Selection (Умный выбор)**:
    *   Использование стрелок для навигации.
    *   `Space` для переключения выбора в группах типа `M-of-N`.
    *   `Enter` для выбора эксклюзивных опций в группах `1-of-N`.
    *   Мгновенная обратная связь о несовместимых комбинациях (если будут реализованы ограничения).

3.  **Live Preview (Предпросмотр)**:
    *   Отображение diff изменений, которые будут внесены в `pyproject.toml`, перед сохранением.

4.  **One-Click Sync**:
    *   Кнопка "Apply & Sync" для сохранения конфига и немедленного запуска `uv sync`.

## Технический стек

*   **Библиотека**: `textual` или `rich` (Python).
*   **Интеграция**: Новая команда `hpm ui` или `hpm interactive`.

## Референсы (2025-2026)

*   **LazyGit/LazyDocker**: Золотой стандарт интерактивности в TUI.
*   **Vercel CLI**: Отличный пример "мастеров" (wizards) настройки проекта.
*   **pnpm interactive**: Позволяет обновлять зависимости, выбирая их из списка.
